## About

This is an implementation of Canny Edge Detector Algorithm with Python

## Prerequisites 

Install OpenCV:
```
npm install --global gulp
```

Install Matplotlib:
```
pip install matplotlib
```

Install NumPy:
```
pip install numpy
```
